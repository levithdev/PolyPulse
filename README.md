# PolyPulse

## Sobre

Projeto simples para aprendizado de consumo de API e gerenciamento de estado em uma aplicação frontend.

A aplicação consome dados do Polymarket e exibe listas de mercados com navegação direta para o site original.

---

## Tecnologias

* React
* TypeScript
* TailwindCSS
* TanStack Query
* Axios
* Zustand
* Browser Notifications API

---

## Funcionalidades

* Feed de mercados
* Lista de mercados com alta movimentação
* Histórico de itens acessados
* Redirecionamento para o Polymarket
* Notificações para novos itens relevantes

---

## Instalação

```bash
npm install
npm run dev
```

---

## Objetivo

* Praticar consumo de API
* Trabalhar com cache e sincronização de dados
* Utilizar gerenciamento de estado simples
* Explorar funcionalidades nativas do navegador
